##### 2. 寄存器定义统一 (Critical Fix)

**问题**: 与 Architecture Spec v1.1 存在关键寄存器位定义不一致

**修正内容**:

| 寄存器 | 修正项 | 修正前 | 修正后 |
|--------|--------|--------|--------|
| CTRL (0x00) | [9]位 | 未定义 | DUAL_RAIL_EN |
| STATUS (0x04) | [4]位 | TIMEOUT_ERR | FAULT_DETECTED |
| STATUS (0x04) | [3:1]位 | 错误标志 | STATE[2:0] |
| INT_EN (0x48) | [0]位 | DONE_EN | ERROR_INT_EN |
| INT_EN (0x48) | [1]位 | ERROR_EN | DONE_INT_EN |
| INT_EN (0x48) | [2]位 | KEY_READY_EN | FAULT_INT_EN |
| INT_STATUS (0x4C) | [0]位 | DONE_STATUS | ERROR_STATUS |
| INT_STATUS (0x4C) | [1]位 | ERROR_STATUS | DONE_STATUS |
| INT_STATUS (0x4C) | [2]位 | KEY_READY_STATUS | FAULT_STATUS |

**参考文档**: Architecture Spec v1.1 (2026-04-01)
