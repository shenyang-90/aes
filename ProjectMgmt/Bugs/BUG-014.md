## Summary

The INT_STAT (Interrupt Status) register at address 0x04C was not functional. It always read as 0 or undefined.

---

## Problem Description

### Register Definition

| Field | Bits | Type | Description |
|-------|------|------|-------------|
| INT_DONE | 0 | RC | Operation complete |
| INT_ERROR | 1 | RC | Error detected |
| INT_FAULT | 2 | RC | Fault detected |
| INT_DMA | 3 | RC | DMA complete |

### Original Implementation (Bug)
```verilog
// apb_if.v / aes_top.v (Bug)
// INT_STAT register not connected to interrupt sources
reg [31:0] int_stat_reg;
always @(posedge clk) begin
    if (apb_rd && addr == 12'h04C)
        prdata <= int_stat_reg;  // Always 0 - never updated!
end
// No logic to set int_stat_reg from interrupt sources
```

**问题：** 中断状态寄存器没有连接到实际的中断源，始终读取为0。

### Fixed Implementation
```verilog
// aes_top.v (Fixed)
// INT_STAT connected to interrupt sources with RC/W1C behavior
reg [3:0] int_stat_reg;

// Set interrupt flags
always @(posedge clk or negedge rst_n) begin
    if (!rst_n) 
        int_stat_reg <= 4'h0;
    else begin
        // Set flags when events occur
        if (op_done && int_en[0]) int_stat_reg[0] <= 1'b1;
        if (error_detected && int_en[1]) int_stat_reg[1] <= 1'b1;
        if (fault_detected && int_en[2]) int_stat_reg[2] <= 1'b1;
        // ...
        
        // RC (Read-Clear) or W1C (Write-1-to-Clear)
        if (apb_wr && addr == 12'h04C)
            int_stat_reg <= int_stat_reg & ~pwdata[3:0];  // W1C
    end
end
```

---

## Root Cause Analysis

1. **未连接中断源**: int_stat_reg没有从done/error/fault信号更新
2. **缺少RC逻辑**: 读取后没有自动清除机制
3. **INT_EN依赖未处理**: 没有检查中断使能位
4. **APB接口不完整**: 写路径没有实现W1C清除

---

## Fix Details

### Changes Made

1. **连接中断源到INT_STAT**
   - INT_DONE <= op_done && INT_EN[0]
   - INT_ERROR <= error_detected && INT_EN[1]
   - INT_FAULT <= fault_detected && INT_EN[2]
   - INT_DMA <= dma_done && INT_EN[3]

2. **实现RC (Read-Clear) 行为**
   - 读取时自动清除已设置的位
   - 或者使用W1C (Write-1-to-Clear)
   - 最终采用: W1C清除方式

3. **添加INT_EN依赖**
   - 只有使能的中断才会设置INT_STAT
   - 符合中断控制器标准行为

4. **完善APB接口**
   - 读路径: 返回int_stat_reg
   - 写路径: W1C清除指定位
