## Root Cause

Iverilog does not support unpacked structs/typedefs. All testcases have been refactored to use parallel arrays:

**Before (Not Supported):**
```systemverilog
typedef struct {
    string name;
    bit [127:0] key;
    ...
} vector_t;
vector_t vectors[5] = '{...};
```

**After (Fixed):**
```systemverilog
reg [127:0] keys [0:6];
reg [127:0] plaintexts [0:6];
reg [127:0] ciphertexts [0:6];

initial begin
    keys[0] = 128'h00010203...;
    plaintexts[0] = 128'h00112233...;
    ciphertexts[0] = 128'h69c4e0d8...;
end
```
