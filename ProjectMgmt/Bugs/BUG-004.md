**Status:** Open

## Summary

GCM (Galois/Counter Mode) implementation is incomplete. Current RTL only performs basic CTR encryption without GHASH authentication.

## Error Details

### Test Failure Log
```

--- Test 5: Key Sensitivity in GCM ---
[FAIL] Different keys should produce different output
  Key 1 output: ...
  Key 2 output: ... (same as Key 1)
```

### Symptoms

1. **AAD Insensitive**: Additional Authenticated Data (AAD) does not affect output
2. **No Tag Generation**: No authentication tag is produced
3. **Missing GHASH**: Galois field multiplication not implemented

## Root Cause Analysis

Current implementation:
```verilog
// Current: Mode 3'd3 = GCM is mapped to CTR
MODE_GCM = 3'd3  // Actually does CTR only
```

Missing components:
1. **GHASH Engine**: Galois field multiplier for authentication tag
2. **GCM Controller**: AAD handling and tag generation state machine
3. **Length Register**: Store AAD length and ciphertext length
4. **Tag Buffer**: 128-bit authentication tag storage

## Expected Behavior (NIST SP 800-38D)

```
GCM_Encrypt(K, IV, P, A):
1. H = E(K, 0^128)                    // Hash subkey
2. Define GHASH function              // GF(2^128) multiply
3. J0 = IV || 0^31 || 1               // Counter pre-processing
4. C = GCTR(K, J0, P)                 // CTR encryption
5. S = GHASH(A || C || len(A) || len(C))  // Authentication tag
6. return (C, MSB_t(S XOR E(K, J0)))  // Ciphertext + Tag
```

## Required Implementation

### 1. New Module: gcm_engine.v
```verilog
module gcm_engine (
    input         clk, rst_n,
    input         gcm_en,
    input  [127:0] hash_subkey_h,    // H = E(K, 0)
    input  [127:0] aad,              // Additional data
    input  [63:0]  aad_len,          // AAD bit length
    input  [127:0] ciphertext,       // Ciphertext input
    input  [63:0]  ct_len,           // CT bit length
    output [127:0] tag,              // Authentication tag
    output         tag_valid
);
    // GHASH computation: GF(2^128) multiplication
    // Reduction polynomial: x^128 + x^7 + x^2 + x + 1
endmodule
```

### 2. Modify mode_controller.v
- Add GCM state machine (AAD_PHASE → CRYPT_PHASE → TAG_PHASE)
- Interface to gcm_engine
